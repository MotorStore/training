框架学习
库文件 http://static.test.gip.happyelements.com/common/lib_0_1.js 备注
broswer function () 获取浏览器版本号，返回值格式为 msie/6.0
$ function (slector) 获取一个或者多个元素 slector格式为 #id .class tagName, 返回值为单个object或者object array
getProp function (obj,prop) 获取obj的属性prop的值
setProp function (obj,prop,value) 将obj的属性prop的值改写为value
show function (obj) 显示元素 obj为#id或者一个object
hide function (obj) 隐藏元素 obj为#id或者一个object
hasClass function (obj,_class) 判断obj是否引用_class的class,返回boolean值
addClass function (obj,_class) 给obj添加样式_class
removeClass function (obj,_class) 将obj的样式_class移除
encode function (c, d, a) 将object c通过连接符d转化为字符串，a为boolean变量，true时对object[key]做encodeURIComponent处理
decode function (f) 将a=1&b=2转化为{a:1,b:2}
guid function () 生成一个以f开头的随机字符串
json2Obj function (astr) 将json格式的字符串（'{a:1,b:2}'）转化为json变量（{a:1,b:2}）
getPara function (url,name) 获取地址url中的参数name的值
setPara function (url,name,value) 将地址url中的参数name的值设置为value，如果url中没有name则为新增，有则覆盖
getHash function (name) 获取地址栏hash中name变量的值
setHash function (name,value) 将地址栏hash中的参数name值设置为value，没有新增，有则覆盖
substitute function (template, data, regexp) 将html模板与数据data自动处理成想用的html代码片段
insertCSS function (link) 往页面中插入地址为link的css样式表
insertJS function (link,callback) 往页面中插入地址为link的js文件，callback为此文件加载完毕的回调
swfObject function (data) 根据数据data生成一段falsh的代码
loadJsonP function (asUrl, callback) 将页面asUrl返回的值作为callback的参数并放入页面的head区域
loadAjax function (option) post方式的ajax, option格式为{url:url,mothed:'post',data:{a:1,b:2},callback:callback}
trim.left function (str) 移除字符串str左边的空格
trim.right function (str) 移除字符串str右边的空格
trim.both function (str) 移除字符串str两端的空格
trim.all function (str) 移除字符串str中所有的空格
array.indexOf function (a, c) 返回字符c在字符串a的位置，返回变量c在数组a中的位置
array.merge function (c, b) 合并两个数组c和b,并且自动合并相同项
array.filter function (a, c) 通过方法c来过滤数组a
array.keys function (c, d) 将object c的所有key值组成数组返回
array.forEach function (c, a, f) 通过方法a处理c的每一项值, 方法的形参为a(c[d], d, c), c为数组时,d为数组的index值，而当c为object时,d为key
